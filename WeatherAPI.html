Check your Temp Weather API

Using API's for the first time

Enter the location that you want to get the temperature for in the Location field. For Dublin, Ireland please write "dublin,ie", else it will show you temperatures for Dublin in America. Once you have chosen your location please press the Submit button.



I have wanted to try playing around with API's for a while now. Weather API's seemed to be a perfect place to start! Github Pages do not allow php files, which made it a little bit more tricky to get it working as I am still a newbie when it comes to javaScript and JQuery.

I think for this being my first go at API's it came out well!
