Hand Gesture Recognition Tool

Software Engineering Project

As part of my Software Engineering Module I had to undertake a 10 week software development project. Together with my team we were tasked with creating a tool that would allow people in the medical field to use their hand as a mouse and interact with the computer interface without physically touching it.

If the website doesn't allow you to view the video click here.

The program:

  • Allows the user to navigate around the screen by pinpointing the mouse to the top of their finger.
  • Allows the user to click the mouse by performing a hand gesture.
  • Has a timer which takes in the users skin colour and background for better detection.
  • Allows you to use your mouse without actually touching the hardware or the screen.

Some things I learned through this project:
  • How to use github through command line.
  • How to use python and OpenCV to create projects like this.
  • Good teamwork is essential for projects like this.
  • Asking for help is much better than struggling on your own.
  • Client communication is not always easy.
